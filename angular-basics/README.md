# Трекер финансов

Веб-приложение для учёта личных финансов: добавление доходов и расходов, просмотр истории транзакций, визуализация статистики по категориям.

## Функциональность приложения

### Форма добавления транзакций

- Выбор вида транзакции (доход / расход) через Taiga UI Block
- Выбор категории из списка (разные списки для доходов и расходов)
- Ввод суммы с валидацией (от 0 до 10 000 000 ₽)
- Выбор даты (нельзя выбрать дату из будущего)
- Необязательный комментарий (до 100 символов) с управляемой валидацией через кастомную директиву
- Валидация всех полей с уникальными сообщениями об ошибках
- Уведомление об успешном сохранении

### История транзакций

- Отображение всех транзакций, отсортированных по дате (от новых к старым)
- Цветовая индикация: зелёный для доходов, красный для расходов
- Кастомный пайп для форматирования суммы с разделением разрядов пробелами, знаком «+» / «−» и символом валюты (₽)
- Тултип с комментарием при наведении на транзакцию
- Анимация появления новых транзакций
- Кнопка редактирования для каждой транзакции

### Статистика

- Кольцевые диаграммы (RingChart) для доходов и расходов с группировкой по категориям
- Интерактивные лейблы внутри диаграммы (категория + сумма при наведении на сектор)
- Легенда с цветными маркерами, совпадающими с секторами диаграммы
- Визуальное разделение доходов и расходов вертикальной чертой

### Редактирование транзакций

- Заполнение формы данными выбранной транзакции
- Подсветка редактируемой транзакции в списке
- Возможность отмены редактирования
- Сохранение изменений в localStorage

### Сохранение данных

- Автоматическое сохранение транзакций в localStorage
- Восстановление данных при перезагрузке страницы

### Ключевые решения

- **Standalone components** — все компоненты standalone, без NgModules
- **Signals** — управление состоянием через Angular Signals (`signal`, `computed`)
- **Реактивные формы** — `FormBuilder`, `FormControl` с валидаторами
- **Two-way binding** — двустороннее связывание для чекбокса комментария (`[(ngModel)]`)
- **Кастомная директива** — `CommentValidatorDirective` динамически добавляет/убирает валидаторы
- **Кастомный пайп** — `TransactionAmountPipe` форматирует сумму в российском формате
- **localStorage** — персистентное хранение транзакций
- **Angular Animations** — анимация появления транзакций в списке

## Установка и запуск

```bash
npm install     -     Установит все необходимые зависимости, указанные в package.json, включая Angular, Taiga UI, NX и все dev-зависимости   (ESLint, Prettier, TypeScript и др.).

npm start       -     Запуск dev-сервера. Приложение запустится на http://localhost:4200.

```

- Node.js v18+
- npm v9+
